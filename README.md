# Game Price Comparer EN-US

## Description

**Game Price Comparer** is a Python CLI application that allows users to search for a game or DLC and quickly find the lowest available price across multiple stores using an external API.

The application provides:
- The exact game or DLC title  
- The store offering the lowest price  
- The regular (average) price  
- A direct link to the store page  

---

## Technologies Used

- Python 3  
- Requests (API consumption)  
- JSON (data handling)  
- Pytest (unit testing)  

---

## Key Learning Outcomes

- Consuming and handling data from an external REST API  
- Working with endpoints and HTTP methods  
- Structuring a project using object-oriented programming (OOP)  
- Writing clean, modular, and testable Python code  
- Solving a real-world problem with a CLI-based solution  

---

## Motivation

This project was built to solve a practical problem: manually searching multiple stores to find the best price for a game is time-consuming and inefficient.

By leveraging the **IsThereAnyDeal** public API, this tool automates the process and delivers clear and useful results through a simple command-line interface.

# Game Price Comparer PT-BR

## Descrição

**Game Price Comparer** é uma aplicação CLI desenvolvida em Python cujo objetivo é consultar e comparar preços de jogos e DLCs em múltiplas lojas digitais, identificando automaticamente a opção com o menor valor disponível por meio do consumo de uma API externa.

A aplicação retorna:
- O título normalizado do jogo ou DLC  
- A loja que apresenta o menor preço no momento da consulta  
- O preço regular (preço médio de mercado)  
- Um link direto para a página do produto na loja correspondente  

---

## Tecnologias Utilizadas

- Python 3  
- Requests (consumo de API REST)  
- JSON (serialização e desserialização de dados)  
- Pytest (testes unitários e validação de funcionalidades)  

---

## Principais Aprendizados Técnicos

- Consumo, interpretação e validação de dados provenientes de uma API REST externa  
- Compreensão e utilização de endpoints e métodos HTTP  
- Estruturação de um projeto utilizando conceitos de Programação Orientada a Objetos (OOP)  
- Desenvolvimento de código modular, legível e testável  
- Implementação de uma solução CLI para resolução de um problema real  

---

## Motivação

O projeto foi desenvolvido com o objetivo de resolver um problema recorrente: a necessidade de consultar manualmente diversas lojas digitais para identificar o menor preço de um jogo, o que é um processo repetitivo e pouco eficiente.

Por meio da utilização da API pública do **IsThereAnyDeal**, a aplicação centraliza essas informações, automatiza a comparação de preços e fornece resultados objetivos por meio de uma interface de linha de comando simples e eficiente.
